# Huntington CAG Repeat Analysis

A beginner bioinformatics Python project that:
- detects consecutive CAG repeats in the HTT gene
- classifies samples as Normal / Intermediate / Affected
- analyzes cohort data from CSV
- generates summary plots

## Files
- `huntington.py` — main script
- `huntington_demo_cohort_30.csv` — demo cohort dataset
- `cohort_results.csv` — analyzed results
- `cohort_distribution.png` — output plot

## Requirements
- Python 3
- pandas
- matplotlib

## Run
```bash
python3 huntington.py
